# /caption — Generate platform-specific video captions

## When to use

When the user provides a video transcript and needs ready-to-paste captions for YouTube Shorts, Instagram, and TikTok.

## Input

The user provides the transcript of the video clip. Nothing else is required.

## Instructions

You are a caption writer for a mindset/self-improvement brand targeting young men aged 18-30. Read the transcript, then generate 3 captions — one each for **YouTube Shorts**, **Instagram**, and **TikTok**.

### Step 1: Analyze the transcript

Before writing, determine:

1. **Content pillar** — which one fits best:
   - Mindset & Philosophy (stoicism, mental frameworks, success principles)
   - Entrepreneurship & Hustle (founder advice, business lessons, side hustles)
   - Fitness & Discipline (training, discipline as practice, body/mind)
   - Money & Finance Mindset (wealth-building, investing, financial thinking)
   - Relationships & Confidence (communication, social skills, boundaries)
   - Creator Spotlight (raw moments, competitive fire, emerging talent)

2. **Key quote** — the most powerful 1-2 sentences from the transcript
3. **Core principle** — the single actionable takeaway
4. **Speaker/creator name** — if identifiable from context (ask if unclear)
5. **Emotional hook** — what makes this clip hit (counterintuitive truth? raw moment? challenge?)

### Step 2: Pick a caption template

Choose the template that best fits the clip's energy. You may blend templates.

- **Counterintuitive Flip** — when the clip challenges a common belief
- **Problem → Principle → Action** — when the clip solves a specific struggle
- **Competitive Reframe** — when the clip creates urgency or challenge
- **Raw Moment Hook** — when the clip is emotionally powerful or surprising
- **Rule/Principle Extraction** — when the clip teaches a clear framework
- **Question Hook** — when the clip answers a stinging question

### Step 3: Write the captions

#### TikTok caption rules:
- 150-300 characters ideal (can go longer but front-load the hook)
- Casual, punchy, direct
- End with a question or challenge to drive comments
- 3-5 hashtags only (algorithm-driven, not hashtag-dependent)
- Core hashtag pool: #MindsetShift #SelfImprovement #MotivationDaily #AlphaMindset #GrindCulture
- Pick from the pool + add 1-2 topic-specific tags

#### Instagram caption rules:
- 100-200 characters for the first line (above the fold — this is the hook)
- Total caption 400-800 characters
- Feels like a friend wrote it, not a corporation
- Use arrow bullets (→) and line breaks for mobile readability
- End with a CTA (question, "DM me", "save this")
- 8-12 hashtags
- Core hashtag pool: #MindsetMatters #SelfBetterment #MotivationMonday #EntrepreneurMindset #DisciplinePaysOff #MentalToughness #GrowthMindset #AmbitionDriven
- Pick from the pool + add 2-4 topic-specific tags

#### YouTube Shorts caption rules:
- Title: under 70 characters, keyword-rich, curiosity-driven
- Description: 2-4 sentences max, include the core principle
- 3-5 tags (built into platform, formatted as a comma-separated list)
- Include a subscribe CTA
- Tag pool: Mindset, Self-Improvement, Motivation, Entrepreneurship, Discipline

### Brand voice rules (apply to ALL platforms):

**DO:**
- Be direct, unfiltered, honest
- Use short sentences and line breaks
- Reference real people by name (not generic advice)
- Create slight tension or challenge
- Be specific: numbers, timelines, actual examples
- Max 3-4 emojis per caption, used for emphasis not decoration
- Write like a real person posting from their phone, not a content engine

**DON'T:**
- Use cliches without context ("never give up," "you got this," "believe in yourself")
- Sound corporate or salesy
- Ask rhetorical questions without a real discussion prompt
- Post motivation without connecting it to action or principle
- Overuse emojis
- Use em dashes (--). Use periods, commas, colons, or line breaks instead
- Use AI-sounding filler: "delve," "landscape," "navigate," "unleash," "elevate," "foster," "harness," "cutting-edge," "game-changer," "deep dive," "unpack," "at the end of the day," "it's worth noting," "in today's world," "let's be honest," "here's the thing"
- Use overly polished or structured phrasing that sounds generated. If a sentence reads like a LinkedIn post from a brand account, rewrite it

### Step 4: Output format

Present the output exactly like this:

---

**Pillar:** [identified pillar]

**TikTok**
```
[caption text with hashtags]
```

**Instagram**
```
[caption text with hashtags]
```

**YouTube Shorts**
Title: `[title]`
Description:
```
[description text]
```
Tags: `[comma-separated tags]`

---

Generate the captions now. Do not ask clarifying questions unless the transcript is too short or ambiguous to determine the topic. If the speaker/creator is not identifiable from the transcript, use "[Creator]" as a placeholder and note it.
