## What You'll Build (DIY Option)

### Claude Code Stack
```
Input: Twitch stream or VOD URL
  ↓
Whisper: Transcribe to text (free, local)
  ↓
PySceneDetect: Find scene boundaries (free, local)
  ↓
Claude API: Analyze transcript for viral moments ($0.05/video)
  ↓
FFmpeg: Extract & render clips (free, GPU-accelerated)
  ↓
Multi-platform APIs: Submit to bounty platforms (free APIs)
  ↓
Output: 20-50 clips, published to 4 platforms, earnings tracked
```

### MCP Server Design
```
mcp_clipper.py
├── transcriber() - Whisper integration
├── scene_detector() - PySceneDetect integration
├── moment_analyzer() - Claude API integration
├── clip_renderer() - FFmpeg orchestration
├── publisher() - Multi-platform distribution
└── monitor() - Earnings aggregation
```

### Rough Code Structure
```python
# Claude can write this in ~100 lines for main orchestration

    def process_vod(self, url):
        vod = download_vod(url)
```
