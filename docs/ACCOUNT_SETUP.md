## Key Differences from Old Structure

### OLD (5 separate emails per platform):
```

clipping-account-2@gmail.com → TikTok Account 2
clipping-account-3@gmail.com → Instagram Account 1
... etc (25 emails total for 5×5 accounts)
```
**Cons:** Email management nightmare (25 Gmail accounts to log into)
**Pros:** Maximum account separation

### NEW (consolidated per platform):
```
tiktok-clipping@gmail.com → TikTok Accounts 1-5 (all linked to same email, different phone numbers)
instagram-clipping@gmail.com → Instagram Accounts 1-5
... etc (4 social + 1 bounty = 5 emails total)
```
**Cons:** Slightly higher detection risk if one account gets banned (same email flag)
**Pros:** 80% easier account management; same safety at 5-account scale

## FAQ

**Q: Won't sharing tiktok-clipping@gmail.com for all 5 TikTok accounts get me banned?**
A: No. Platforms detect account networks by behavior (posting patterns, engagement), not email. Separate phone numbers per account + staggered creation + varied posting times = safe. Email consolidation is fine at 5-account scale.

**Q: Should I use VPNs or proxies?**
A: Yes, for medium/high-scale (10+ accounts). At 5-account scale with staggered creation and humanized posting, proxies optional but recommended ($50-150/month). See 02_BOT_AVOIDANCE_PLAYBOOK.md for details.

**Q: Can I use the same payment method for all accounts?**
A: Yes, for bounty platforms and social media accounts. Platforms flag identical posting patterns + metadata, not shared payment cards. One Privacy.com card works for all.

**Q: What if one TikTok account gets banned?**
A: You still have 4 TikTok accounts posting. That's 80% capacity. Wait 60 days, create a new account using same email (tiktok-clipping@gmail.com) with a new phone number (e.g., +1-XXX-0003f). Simple recovery.

**Q: Should I create all 5 accounts per platform at once?**
A: No. Stagger by 48+ hours per account. See timeline above.
